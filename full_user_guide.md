# Baku任务系统 - 完整使用指南

## 1. 系统概述

Baku任务系统是一个任务驱动的积分奖励系统，用户可以通过完成特定任务（如添加机器人到群组、关注Twitter等）来获得积分。

## 2. 首次使用流程

### 步骤1：连接Telegram账户
1. 打开Telegram应用
2. 搜索并找到 `@baku_news_bot`
3. 向机器人发送任意消息（如"hi"、"hello"或使用`/start`命令）
4. 这个操作会将您的Telegram用户ID与Baku系统关联

### 步骤2：访问任务系统
1. 返回Baku网站 (baku.builders)
2. 点击主页上的"Complete Tasks & Earn Points"按钮
3. 系统会自动检测您的Telegram连接状态
4. 如果已连接，您将直接进入个性化任务页面

### 步骤3：完成任务获取积分
1. 浏览可用任务列表
2. 点击"Claim Task"领取任务
3. 按照任务说明完成相应操作
4. 系统会自动验证并发放积分

## 3. 验证机制详解

### 自动验证流程
- 当您首次与@baku_news_bot交互时，Telegram会发送webhook到Baku服务器
- 服务器捕获您的Telegram用户ID并存储在会话中
- 之后访问任务系统时，系统会识别您的身份
- 您将看到针对您个人的个性化任务和积分统计

### 会话管理
- 您的Telegram身份信息存储在会话中
- 会话有一定时效性，长时间不活动后可能需要重新连接
- 重新与机器人交互即可恢复连接

## 4. 可用任务类型

- **添加机器人到群组** (+100积分)：将@baku_news_bot添加到您的Telegram群组
- **关注Twitter** (+100积分)：关注@Baku_builders Twitter账号
- **加入Telegram频道** (+50积分)：加入官方Baku频道
- **转发推文** (+75积分)：转发来自@Baku_builders的推文

## 5. 故障排除

### 如果无法访问任务页面
1. 确认已向@baku_news_bot发送过消息
2. 尝试刷新网站页面
3. 清除浏览器缓存后重试
4. 确保在同一个浏览器中进行操作

### 如果积分未更新
- 积分更新可能需要几分钟时间
- 尝试刷新任务页面
- 某些任务可能需要手动验证

## 6. 安全注意事项

- 您的Telegram身份仅用于系统识别
- 个人信息受到保护
- 仅存储必要的用户标识符
- 积分数据公开显示在排行榜上，但不泄露敏感信息

## 7. 技术支持

如果遇到持续的技术问题：
1. 检查网络连接
2. 尝试使用不同浏览器
3. 确认Telegram机器人正常运行
4. 联系Baku团队获取支持
